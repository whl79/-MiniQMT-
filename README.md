# -MiniQMT-
MiniQMT是一个非常友好的A股量化工具，通过中文方便设定全局交易参数、本地股票池和本地交易策略，可以实现程序化自动交易。同时，可以查看持仓、成交和委托的执行情况。还可以查看实时的主要指数行情。下载后此文件后，开启MiniQMT账号，并在Dist文件夹中找到懂量化MiniQMT稳定版(新版).exe，左键双击启动程序后，通过设定QMT路径、账号后，就可以运行此程序。
