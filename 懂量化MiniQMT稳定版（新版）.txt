懂量化MiniQMT稳定版（新版）是一款基于Python与PySide6开发的专业量化交易系统，深度整合QMT（量化交易平台）的核心交易能力，为用户提供一套完整、高效的自动化交易解决方案。系统采用全中文操作界面，简洁直观、易于上手，助力用户快速掌握量化交易操作流程。

•	支持多策略并行管理，可灵活切换本地自定义策略与平台预设策略

•	图形界面简洁直观，操作逻辑清晰，降低量化交易入门门槛

5. 功能说明
5.1 通达信公式引擎
系统内置完整的通达信公式解析引擎，全面兼容通达信策略语法，支持以下核心功能：
•	变量赋值：VAR:=表达式（支持各类数值、逻辑表达式）
•	内置函数：涵盖EMA、MA、SMA、MACD、CROSS、REF、HHV、LLV等常用指标函数
•	逻辑运算：支持AND（与）、OR（或）、NOT（非）等逻辑操作
•	信号输出：通过ENTERLONG（买入信号）、EXITLONG（卖出信号）定义交易触发条件
5.2 策略管理
•	双策略来源支持：可灵活选用本地自定义策略或平台预设策略
•	内置策略模板：涵盖多指标组合策略、MACD策略、KDJ策略、RSI策略等经典量化策略，可直接使用或修改
•	策略编辑器：支持用户自定义策略编写、修改与保存，适配通达信语法，上手便捷
5.3 交易执行
•	交易方式灵活：支持按交易金额或按持股数量两种下单方式
•	规则合规校验：内置A股T+1交易规则验证，避免违规操作
•	自动下单撤单：根据策略信号自动执行下单、撤单操作，无需人工干预
•	执行结果反馈：订单执行状态实时反馈，确保用户及时掌握交易情况
5.4 风险管理
•	止损功能：当个股跌幅达到预设止损幅度时，系统自动触发卖出操作，控制亏损
•	止盈功能：当个股涨幅达到预设止盈幅度时，系统自动触发卖出操作，锁定收益
•	仓位控制：当账户总仓位超出预设最大仓位比例时，自动减仓，规避过度持仓风险
5.5 自动功能
•	自动逆回购：15:05自动执行逆回购操作（仅支持实盘账号，模拟账号请勿勾选，避免程序运行出现异常）
•	自动数据下载：15:10自动下载当日行情数据，为后续策略分析提供支撑
•	非交易时间交易（可选）：勾选后，非交易时段若触发买入/卖出信号，需手动在QMT端取消所有委托，避免无效委托或次日交易日出现交易执行的现象
5.6 数据管理
•	实时行情获取：同步获取A股实时行情数据，延迟低、准确性高
•	历史K线下载：支持手动/自动下载历史K线数据，满足策略回测需求
•	数据缓存机制：对常用数据进行本地缓存，大幅提升系统运行效率，减少网络请求
6. 使用步骤
6.1 启动系统
运行“懂量化MiniQMT稳定版（新版）.exe”文件（若为.py文件，可通过Python环境运行）
首次登录需填写用户基础参数及相关配置，配置完成后务必点击“保存配置”，否则配置无法生效
系统自动完成初始化，加载预设配置及基础数据，等待后续操作
6.2 配置QMT账户
在左侧配置区域，准确填写QMT安装路径及对应交易账号信息
点击“连接QMT账户”按钮，发起连接请求
确认连接状态显示为“● 已连接”，即表示QMT账户连接成功
6.3 配置股票池
选择股票池来源（本地文件或平台股票池）
若选择本地股票池：点击“浏览”按钮，选中通达信板块文件（.blk格式），完成加载（请自行学习如何安装通达信软件和设置通达信条件选股公式）
若选择平台股票池：先完成量化平台连接，再点击“刷新”按钮，选择目标股票池即可
6.4 配置交易时间段
勾选需要启用的交易时间段（上午盘/下午盘，可同时勾选）
设置具体的交易开始时间与结束时间，需符合A股交易时段规范
根据需求，可选启用“自动逆回购”“自动下载行情”功能
6.5 选择交易策略
选择策略来源（本地自定义策略或平台预设策略）
若选择本地策略：从下拉菜单中选中目标内置策略，可直接使用
若选择平台策略：先完成量化平台连接，再从平台策略列表中选择目标策略
点击“策略编辑”按钮，可对选中策略进行自定义修改（需遵循通达信语法）
策略配置完成后，点击“保存配置”按钮
重启程序，确认配置已成功应用
6.6 开始交易
点击“连接平台账号”，准确输入服务器地址、用户名及密码，完成平台连接
确认QMT账号已成功连接（状态显示“已连接”）
选择选股策略方式，匹配已配置的股票池
确认当前启用的交易策略无误
点击“刷新股票池”，确保股票池数据为最新状态
点击“开始交易”按钮，系统启动交易逻辑
交易过程中，可在右侧日志区域查看实时执行进度及相关信息
通过中间监控区域，实时查看持仓明细、订单状态及成交记录
6.7 停止交易
点击底部“停止交易”按钮，系统将立即终止交易逻辑
停止交易后，QMT账户与量化平台账户将保持连接状态，便于后续再次启动交易
策略编写指南
7.1 策略语法
系统全面支持通达信公式语法，核心语法规则如下：
•	变量定义：采用 VAR:=表达式 格式，用于定义策略中的各类变量（如均线、成交量等）
•	条件判断：使用 IF(条件, 真值, 假值) 格式，实现逻辑判断与结果返回
•	信号输出：
￮	ENTERLONG:=条件：定义买入信号触发条件
￮	EXITLONG:=条件：定义卖出信号触发条件
7.2 内置函数
系统集成各类常用通达信函数，覆盖指标计算、逻辑判断等场景，核心函数分类如下：
•	移动平均类：MA（简单移动平均）、EMA（指数移动平均）、SMA（加权移动平均）
•	常用指标类：MACD（指数平滑异同移动平均线）、KDJ（随机指标）、RSI（相对强弱指标）、LWR（威廉指标）、BBI（多空指标）、MTM（动量指标）
•	逻辑判断类：CROSS（金叉/死叉）、REF（引用前期数据）、HHV（最高价）、LLV（最低价）、COUNT（计数）、BARSLAST（上次条件成立距离当前周期数）
•	数学运算类：ABS（绝对值）、MAX（最大值）、MIN（最小值）、SUM（求和）、STD（标准差）
7.3 策略示例
7.3.1 选股公式示例
均线多头排列选股（适用于筛选趋势向上的个股）
plain text
MA5:=MA(CLOSE,5);  // 5日均线
MA10:=MA(CLOSE,10);  // 10日均线
MA20:=MA(CLOSE,20);  // 20日均线
MA60:=MA(CLOSE,60);  // 60日均线
// 均线多头排列条件：短期均线依次高于长期均线
MA多头排列:=MA5>MA10 AND MA10>MA20 AND MA20>MA60;
// 成交量放大条件：当日成交量较前一日放大20%以上
量能放大:=VOL>REF(VOL,1)*1.2;
// 选股信号：同时满足均线多头排列和量能放大
XG:=MA多头排列 AND 量能放大;
MACD金叉选股（适用于筛选短期反弹个股）
plain text
DIF:=EMA(CLOSE,12)-EMA(CLOSE,26);  // MACD指标DIF线
DEA:=EMA(DIF,9);  // MACD指标DEA线
MACD:=(DIF-DEA)*2;  // MACD柱状线
// MACD金叉条件：DIF线向上穿越DEA线，且DIF线在零轴下方（低位金叉）
MACD金叉:=CROSS(DIF,DEA) AND DIF<0;
// 成交量放大条件：当日成交量较前一日放大50%以上
量能放大:=VOL>REF(VOL,1)*1.5;
// 选股信号：同时满足MACD金叉和量能放大
XG:=MACD金叉 AND 量能放大;
7.3.2 交易策略示例
多指标交易策略（结合MACD、KDJ、RSI三大指标，提升信号可靠性）
plain text
// MACD指标判断
DIF:=EMA(CLOSE,8)-EMA(CLOSE,13);
DEA:=EMA(DIF,5);
A1:=IF(DIF>DEA,1,0);  // MACD金叉状态标记为1，否则为0
MACD金叉:=IF(A1>0,1,0);
// KDJ指标判断
RSV1:=(CLOSE-LLV(LOW,8))/(HHV(HIGH,8)-LLV(LOW,8))*100;
K:=SMA(RSV1,3,1);
D:=SMA(K,3,1);
A2:=IF(K>D,1,0);  // KDJ金叉状态标记为1，否则为0
KDJ金叉:=IF(A2>0,2,0);
// RSI指标判断
LC:=REF(CLOSE,1);
RSI1:=(SMA(MAX(CLOSE-LC,0),5,1))/(SMA(ABS(CLOSE-LC),5,1))*100;
RSI2:=(SMA(MAX(CLOSE-LC,0),13,1))/(SMA(ABS(CLOSE-LC),13,1))*100;
A3:=IF(RSI1>RSI2,1,0);  // RSI金叉状态标记为1，否则为0
RSI金叉:=IF(A3>0,3,0);
// 综合信号：三大指标金叉状态求和
A7:=(A1+A2+A3);
// 交易信号：至少两个指标金叉则触发买入，不足一个则触发卖出
ENTERLONG:IF(A7>=2,1,0);
EXITLONG:IF(A7<1,1,0);
趋势跟踪策略（基于均线趋势，捕捉中长期上涨行情）
plain text
// 长期趋势判断：20日均线高于60日均线，视为趋势向上
MA20:=MA(CLOSE,20);
MA60:=MA(CLOSE,60);
趋势向上:=MA20>MA60;
// 短期买入信号：5日均线向上穿越10日均线（短期金叉）
MA5:=MA(CLOSE,5);
MA10:=MA(CLOSE,10);
短期金叉:=CROSS(MA5,MA10);
// 卖出信号：10日均线向下穿越5日均线（短期死叉）
短期死叉:=CROSS(MA10,MA5);
// 交易信号：趋势向上且出现短期金叉买入，出现短期死叉卖出
ENTERLONG:IF(趋势向上 AND 短期金叉,1,0);
EXITLONG:IF(短期死叉,1,0);
8. 常见问题与解决方案
8.1 连接问题
•	QMT连接失败：检查QMT安装路径是否填写正确，确认QMT交易端已正常启动，且账号无登录异常
•	平台连接失败：核对服务器地址、用户名及密码是否准确，检查网络连接是否稳定，排除网络防火墙拦截
•	网络异常：确认网络连接正常，可尝试重启路由器或切换网络，确保行情数据与交易指令可正常传输
8.2 交易问题
•	无法下单：检查当前是否处于A股交易时段（9:30-11:30、13:00-15:00），确认账户资金充足、个股无停牌或限购限制
•	订单未成交：检查下单价格设置是否合理（如买入价低于当前市价、卖出价高于当前市价），确认个股市场流动性充足，避免挂单价格偏离市价过多
•	进度条无更新：重启懂量化MiniQMT系统及QMT交易端，清除缓存后重新连接，一般可恢复正常
8.3 策略问题
•	策略无信号：检查策略语法是否正确（有无语法错误、函数使用是否规范），确认行情数据完整，可尝试调整策略参数或增加信号过滤条件
•	策略信号频繁：优化策略参数（如调整均线周期、放大成交量阈值），增加额外过滤条件（如趋势判断、仓位控制），减少无效信号
8.4 数据问题
•	行情数据获取失败：检查网络连接，尝试手动点击“下载行情数据”功能，排除网络波动或平台数据接口异常
•	历史数据缺失：使用系统“手动下载”功能，选择对应时间周期的历史数据进行补充下载，确保策略回测与执行的数据完整性
